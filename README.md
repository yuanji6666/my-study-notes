# 📝 这个仓库是学习笔记的备份

# 笔记记什么?

## 1. 记“新旧知识的连接点”

不要记作者说了什么，要记作者的话让你**联想**到了什么。

- **反直觉点：** 哪些内容打破了你原有的认知？（比如：“我原以为A导致了B，但作者证明其实是C。”）
    
- **类比点：** 这个新概念和你已知的哪个概念很像？（比如：“这个经济学原理听起来就像我打游戏时的补兵策略。”）
    
### 2. 记“信息的骨架”（逻辑逻辑还是逻辑）

与其抄录优美的长难句，不如拆解作者的推导过程。

- **逻辑链条：** 核心观点 → 论据1、2、3 → 结论。
    
- **视觉化：** 用箭头、流程图或思维导图代替整段的文字。如果你能用一个公式或简图概括一章的内容，你就已经彻底消化它了。
    
### 3. 记“行动导向” (Actionable Insights)

这是最有价值的部分。问自己：**“这段话对我有什么用？”**

- **应用场景：** “下次我写策划案时，可以用这个模型。”
    
- **待办清单：** “去查一下文中提到的那个参考文献。”
    
- **个人感悟：** “我之前的失败，可能就是因为没注意到这一点。”
    
## 推荐：两个立竿见影的笔记法

| **方法**     | **核心逻辑** | **操作要点**                                                  |
| ---------- | -------- | --------------------------------------------------------- |
| **费曼技巧**   | 以教促学     | 关上书，尝试用**自己的语言**（最好是说给8岁小孩听的白话）把刚才看的内容写下来。写不出的地方才是你没懂的。   |
| **康奈尔笔记法** | 强制思考     | 将页面分为：**笔记区**（简明记录）、**线索区**（提炼关键词/问题）、**总结区**（课后用一两句话概括）。 |

## 💡 一个思维转变的小建议

**“笔记是写给未来的自己看的。”**

想象一下，一个月后的你打开这本笔记，他没有时间重新看一遍原著，他只想知道：

1. 这东西最核心的**干货**是什么？
    
2. 我当时为什么觉得它**重要**？
    
3. 我该怎么**用**它？
